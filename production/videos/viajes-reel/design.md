# Design: Viajes Reel

## Palette
- Background: #0D0D0D
- Accent: #E8C547
- Text primary: #FFFFFF
- Text secondary: rgba(255,255,255,0.75)
- Overlay: linear-gradient(to top, rgba(0,0,0,0.80) 0%, rgba(0,0,0,0.30) 50%, rgba(0,0,0,0.10) 100%)

## Typography
- Display font: Inter, weight 800
- Body font: Inter, weight 400
- Tag font: Inter, weight 500, letter-spacing 0.15em, uppercase

## Motion
- Ken Burns: subtle zoom + pan over 3.6s per scene (scale 1.08 → 1.18)
- Entrances: power3.out, 0.7s
- Scene transitions: crossfade 0.6s
- Text stagger: 0.15s between elements

## Format
- Dimensions: 1080x1920 (vertical, Instagram Reels)
- Duration: 15s
- FPS: 30
