# Clase

## Objetivo
Preparar, estructurar y dar seguimiento al módulo de agentes del máster de IA Generativa en Evolve. Que cada sesión sea práctica, coherente con las anteriores y que los alumnos salgan con algo construido.

## Inputs
- Número o tema de la sesión a preparar
- Duración disponible y formato (teórica / práctica / mixta)
- Estado del módulo: días impartidos hasta ahora
- Feedback de sesiones anteriores (si existe)

## Herramientas
- `skills/clase.md` — proceso detallado
- Web search — para referencias técnicas actualizadas
- `mcp__claude_ai_Google_Drive__*` — materiales en Drive
- `mcp__claude_ai_Google_Calendar__*` — fechas de sesiones

## Pasos

### Preparar una sesión
1. Identificar tema y número de sesión (X/10)
2. Revisar qué se cubrió en sesiones anteriores para no repetir ni saltar pasos
3. Definir el objetivo de la sesión: ¿qué sabe hacer el alumno al terminar?
4. Estructurar en bloques de ~20 min:
   - Bloque 1: concepto clave + ejemplo real
   - Bloque 2: demo o walkthrough
   - Bloque 3: ejercicio práctico
   - Cierre: pregunta de reflexión o reto para casa
5. Preparar código de ejemplo ejecutable (no pseudocódigo)
6. Añadir versión extendida del ejercicio para los más rápidos

### Crear ejercicio práctico
1. Definir el resultado concreto (qué construye el alumno)
2. Calcular el tiempo — asequible para el 80% del grupo en ese tiempo
3. Dar scaffolding mínimo: estructura base, no solución completa
4. Incluir criterios de éxito claros

### Seguimiento del módulo
1. Actualizar contador de días impartidos
2. Identificar temas cubiertos y pendientes
3. Si hay desvío de ritmo: proponer ajuste con opciones concretas
4. Si hay feedback negativo de alumnos: señalar el tema que necesita refuerzo

## Edge cases
- Si no se sabe el nivel del grupo en un tema concreto: preparar en dos capas (básico + avanzado)
- Si falta tiempo en sesión: priorizar ejercicio práctico sobre teoría adicional
- Si un concepto técnico es muy reciente: hacer web search antes de presentarlo
- Si el proyecto final no está definido: preparar sesión asumiendo que el alumno construirá su primer agente funcional
